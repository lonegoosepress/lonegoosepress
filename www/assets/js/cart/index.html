Stripe and Vanilla JS Shopping Cart

Use vanilla client side javascript to build and manage a simple shopping cart that passes off an order to Stripe.js for further processing.

Anatomy of a Cart

├── cart │ ├── add item │ │ ├── name │ │ ├── id │ │ ├── price │ │ └── quantity │ ├── summary │ │ ├── currency │ │ ├── tax rate │ │ ├── tax │ │ ├── shipping │ │ ├── subtotal │ │ ├── total │ │ └── items │ └── checkout │ │ ├── total │ │ ├── item summary │ │ ├── billing info │ │ └── shipping info └────

Usage

var myCart = new Cart(options)

Options is an object with the following defaults:

{
  currency = '$',
  taxRate = 0.00,
  tax = 0.00,
  baseShipping = 0.00,
  shipping = 0.00,
  subtotal = 0.00,
  total = 0.00,
  items = []
}

Add to Cart

Add an item to the cart object. Set the data attributes as needed.

<a href="#" class="js-add-to-cart"
  data-title="Widget Thing"
  data-price="29.99"
  data-quantity="1"
  data-id="uniqueid">Add Item to Cart</a>
